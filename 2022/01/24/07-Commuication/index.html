寄存器

在全双工模式下，需要用到下列寄存器

  • SCICTL1：SCI控制寄存器1
    • 6：RXERRINTENA：接收错误中断使能
      • 0：禁用
      • 1：启用
    • 5：SWRESET：SCI软件重置
      初始化SCI状态机和状态标志
    • 3：TXWAKE：发送唤起模式选择
    • 2：SLEEP：SCI睡眠使能
      • 0：禁用
      • 1：启用
    • 1：TXENA：SCI发送使能
      • 0：禁用
      • 1：启用
    • 0：RXENA：SCI接收使能
      • 0：禁用
      • 1：启用
        在初始化过程中通常仅启用发送与接收，故写入0x0003
        在全部初始化结束后，可通过写入0x0023来重置SCI状态
  • SCICTL2：SCI控制寄存器2
    • 7：TXRDY：发送缓冲就绪标志
      0：SCITXBUF满
      1：SCITXBUF空，可以接收下一个字符
    • 6：TXEMPTY：发送空标志
      0：发送缓冲与发送移位寄存器中均有数据
      1：发送缓冲与发送移位寄存器中均无数据
    • 1：RXBKINTENA：RXRDY，BRKDT中断使能
      控制由SCIRXST.RXRDY(6)与SCIRXST.BRKDT(5)标志位产生的中断
      • 0：禁用
      • 1：启用
    • 0：TXINTENA：TXRDY中断使能
      控制由SCICTL2.TXRDY(7)产生的中断
      • 0：禁用
      • 1：启用

  • SCIFFCT：FIFO控制寄存器

  • SCICCR：通信控制寄存器

    • 7：STOPBITS：停止位数量
      • 0：1位
      • 1：2位
    • 6：PARITY：奇偶校验模式选择
      • 0：奇校验
      • 1：偶校验
    • 5：PARITYENA：奇偶校验使能位
      • 0：禁用
      • 1：启用
    • 4：LOOPBKENA：回环模式使能位
      • 0：禁用
      • 1：启用
    • 3：ADDRIDLE_MODE：不知道是什么使能位
      • 0：禁用
      • 1：启用
    • 2-0：SCICHAR：数据长度
      • 0-7：数据长度1-8
        为了配置8N1，应在该寄存器中写入0x0007

  • SCIHBAUD-SCILBAUD：波特率

    这两个16bit寄存器均只有第八位启用，高八位均保留，拼成一个16位寄存器

  • SCIFFTX：FIFO发送寄存器

    • TXFFST：发送FIFO状态
  • SCITXBUF：发送缓冲寄存器

  • TXSHF：发送移位寄存器

  • SCIFFRX：FIFO接收寄存器

  • SCIRXBUF：接收缓冲寄存器
  • RXSHF：接收移位寄存器

配置

GPIO

为了防止信号尖刺，应当先配置GPyGMUX，然后再配置GPyMUX。对与输入信号，GPIO应当通过GPxQSELn被配置到异步模式

中断

SCICTL2.TXRDY指示发送中断SCIRXST.RXRDY与SCIRXST.BRKDT指示接收中断，即使中断没有被启用，他们也会正常工作，表示目前的状态。

发送与接收有这不同的中断使能标志，同时，发送与接收也有着独立的中断向量，接受的中断向量优先级始终高于发送

当
• If the RX/BK INT ENA bit (SCICTL2, bit 1) is set, the receiver peripheral interrupt request is asserted
when one of the following events occurs:
– The SCI receives a complete frame and transfers the data in the RXSHF register to the SCIRXBUF
register. This action sets the RXRDY flag (SCIRXST, bit 6) and initiates an interrupt.
– A break detect condition occurs (the SCIRXD is low for ten bit periods following a missing stop bit).
This action sets the BRKDT flag bit (SCIRXST, bit 5) and initiates an interrupt.
• If the TX INT ENA bit (SCICTL2.0) is set, the transmitter peripheral interrupt request is asserted
whenever the data in the SCITXBUF register is transferred to the TXSHF register, indicating that the
CPU can write to SCITXBUF; this action sets the TXRDY flag bit (SCICTL2, bit 7) and initiates an
interrupt.

波特率计算

SCI的时钟信号由SYSCLK经Low Speed Prescaler分频后的LSPCLK提供后经由16bit baud-select寄存器计算后得到波特率

其中$\frac{LSPCLK}{16}$是最高的波特率
